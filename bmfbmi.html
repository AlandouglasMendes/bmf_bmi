A bmf and bmi index calculator made using html/css/javascripting

This is a web app written to calculate both body mass index* and body fat mass percentage*

*I have limited acces to limited databases, so by the moment, I just have data for female and male adults. I lack data from transgender adults, and I would be very happy to add them, if you could share from some source to be analyzed(I'm not really into doing it for children. Since theyr body is changing so fast, i don't think these measures would be useful to them without a proper healthh profession).

This app is not a doctor, is just a source of tips! Keep that in mind!

Please insert data on the respective textboxes below and mark measuring accordingly. Use the converters if you have values on imperial system. The values will show automatically when you set all fields properly. please fill biological sex field last.
